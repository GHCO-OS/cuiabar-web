# Arquitetura

## Stack principal

- React 18
- Vite 5
- TypeScript
- Tailwind CSS
- Cloudflare Pages
- Cloudflare Workers
- Cloudflare D1
- Cloudflare KV
- Workers AI

## Núcleo

- `src/` para interfaces
- `worker/` para backend e integrações
- `functions/` para comportamento do site público
- `migrations/` para schema
